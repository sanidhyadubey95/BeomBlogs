Publishing checklist:
  1. Save the Markdown as in your posts/ folder.
  2. Open posts/index.json and paste the JSON entry into the array.
  3. If you used a visualization, place that HTML in visualizations/.
  4. If you added a cover image URL, make sure the image is accessible.
  5. Commit and push to GitHub — your post goes live!
