About

Hi! I'm a curious mind writing about things I find interesting — mathematics, code, data visualizations, and ideas worth exploring.

This blog is built entirely with static HTML, CSS, and JavaScript. No server, no database — just files on GitHub Pages. Posts are written in Markdown and rendered in the browser.

What you'll find here

  • Technical writing with interactive visualizations
  • Mathematical explorations with rendered equations
  • Data essays with embedded charts and graphs
  • Occasional personal reflections

How this site works

Posts are written in Markdown and stored as .md files in the posts/ folder. The posts/index.json file keeps track of all posts. Interactive visualizations live in the visualizations/ folder as self-contained HTML files that get embedded as iframes.
