About

Hi — I write about things I find genuinely interesting: mathematics, code, data, and the ideas that live at their intersections.

This blog is built entirely with static HTML, CSS, and JavaScript. No server, no database, no build step — just files on GitHub Pages. Posts are written in Markdown and rendered in the browser.

What you'll find here

  • Technical writing with live interactive visualizations
  • Mathematical explorations with rendered equations (LaTeX via KaTeX)
  • Data essays with embedded charts and graphs
  • Occasional longer essays on ideas worth examining

How this site works

Posts are .md files in the posts/ folder. A posts/index.json registry lists them all. Interactive visualizations are self-contained HTML files in visualizations/, embedded as iframes wherever a post needs them.

Get in touch

Find me on GitHub or Twitter/X. I read everything.
